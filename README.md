# group5-practicum

Tugas Praktikum 2 - Kelompok (Week 7): implementasi Queue, Stack, dan Linked List dalam Java.

## Requirement

- JDK 11 atau lebih baru (sudah ditest pakai JDK 21).

## Compile

Git Bash / Linux / macOS:

```bash
javac -d bin src/*.java
```

PowerShell / Command Prompt:

```powershell
javac -d bin src\*.java
```

## Run

```bash
java -cp bin Main
```

Setelah dijalankan akan muncul menu utama:

```text
=== MENU UTAMA ===
1. Sistem Manajemen Antrean (Queue)
2. Fitur Undo/Redo Editor Teks (Stack)
3. Manajemen Data Mahasiswa (Linked List)
0. Keluar
```

Ketik angka pilihan lalu ikuti instruksi di layar. Pilih `0` untuk kembali ke menu sebelumnya atau keluar dari program.
